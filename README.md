# Ruby Capstone Project: Stockify 🛅🛅

>  A console app that will help you to keep a record of different types of things you own: books, music albums, movies, and games. Based on UML class diagrams, the data is stored in JSON files.

## Getting Started

This repository includes files with Ruby that can be used to test the created main classes from the app

Clone this link into your local device.
